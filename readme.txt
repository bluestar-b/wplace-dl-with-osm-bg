simple wplace downloader and stitch with osm bg

Installation
1. Make sure you have Python 3 and pip installed.
2. Install pipenv: pip install pipenv
3. Install dependencies: pipenv install
4. Run the script: pipenv run python3 main.py <target_name>


Usage:
pipenv run python3 main.py <target_name>

Example:
pipenv run python3 main.py testmeowmeow
